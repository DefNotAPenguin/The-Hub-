Minecraft Launchers

Alright, so this will get a bit weird.

Some weird stuff happened with some of these, I won't get too in-depth with it (for the sake of privacy and stuffs), but with some Google-fu you should easily be able to find this stuff.

I just don't want to get into it here, therefore there won't be any links to any of these on this page.

Alright let's start this list already.

There's gonna be a "Pros" list and a "Cons" list.

These are entirely subjective and my personal opinion, you might have different opinions.

Some of these "Cons" aren't even that bad, just simply not my thing.

Curseforge

Curseforge's own launcher works perfectly fine. It has direct install buttons from the browser to the app, so if that's important to you then go ahead and use it.

Pros

  • Direct install button from the browser
  • UI is "fine"
  • Easily the most community support
  • Overwolf app
  • App is made by the people that have the biggest mod hosting website on the internet
  • One button install for modpacks

Cons

  • UI is overly done. Could be a lot simpler
  • It is an Overwolf App, so you would also need to download Overwolf
  • No Linux Version that supports Minecraft
  • Closed Source

MultiMC

MultiMC is an open-source launcher aimed towards people that imply don't enjoy using Curseforge for one reason or another.

Pros

  • Really straight to the point UI
  • Can install from Curseforge, Modrinth, Technic, FTB, and ATLauncher
  • Can add mods to instances by the masses
  • Give direct link to mods that cannot be automatically installed
  • Source code is on GitHub
  • Linux support (kind of)
  • Decently sized community

Cons

  • Has had disagreements with other launcher devs
  • Is a "stable" release rather than cutting edge
  • Some mods cannot be automatically installed from Curseforge, requiring manual installs of all those mods
  • Linux users basically couldn't add their account due to the dev team refusing to allow Microsoft account verification, which was needed to play Minecraft

PolyMC

PolyMC is a fork of MultiMC, so it essentially has a lot of the same features.

Pros

  • Same features as MultiMC
  • Very slightly better Linux support
  • Source code is also on GitHub

Cons

  • Dev team is now only one very mean person. I do not recommend using Poly.
  • Basically unusable anyways as account verification doesn't work
  • Code of Conduct was deleted

Prism Launcher

Prism Launcher is also a fork of MultiMC, made by the old team of PolyMC, after being kicked out by the PolyMC dev. So far they seem to be good people, but time will tell. Go to their website, to the wiki, and read the FAQ.

Pros

  • Same features as PolyMC and MultiMC
  • More of a "cutting edge" release with brand new features being tested
  • Source code is on GitHub
  • Code of Conduct exists
  • Has direct support for basically every console, from Windows and Linux, to even the Steam Deck and Nintendo Switch (on Linux)
  • Code of Conduct makes absolute sure that everyone will be treated properly, regardless of any differences in any way

Cons

  • Still requires manual installs, but has an open all links button to speed up the process
